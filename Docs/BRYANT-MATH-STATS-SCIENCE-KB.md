# 4. SUBJECT KNOWLEDGE — CALCULUS I

**Course**: MATH 121 — Calculus and Analytic Geometry I

## 4.1 Limits

### Definitions
- **Limit**: The value that f(x) approaches as x approaches a value c. Written: lim(x->c) f(x) = L
- **One-sided limits**: lim(x->c-) f(x) (left), lim(x->c+) f(x) (right)
- **Limit exists** if and only if both one-sided limits exist and are equal

### Key Formulas
- **Limit Laws**: Sum, difference, product, quotient, power, root laws
  - lim[f(x) + g(x)] = lim f(x) + lim g(x)
  - lim[f(x) * g(x)] = lim f(x) * lim g(x)
  - lim[f(x)/g(x)] = lim f(x) / lim g(x), provided lim g(x) != 0
- **Squeeze Theorem**: If g(x) <= f(x) <= h(x) near c, and lim g(x) = lim h(x) = L, then lim f(x) = L
- **Special limits**:
  - lim(x->0) sin(x)/x = 1
  - lim(x->0) (1 - cos(x))/x = 0
  - lim(x->inf) (1 + 1/n)^n = e

### Key Terms
Indeterminate form (0/0, inf/inf), continuity, removable discontinuity, jump discontinuity, infinite discontinuity, asymptote, end behavior

### Worked Example
Find lim(x->2) (x^2 - 4)/(x - 2).
Direct substitution gives 0/0 (indeterminate). Factor: (x+2)(x-2)/(x-2) = x+2. So lim = 2+2 = 4.

### Common Misconceptions
- "If f(c) is undefined, the limit doesn't exist" — FALSE. Limits depend on nearby behavior, not the value at the point.
- Confusing "limit equals infinity" with "limit exists" — a limit of infinity means the limit does not exist (in the finite sense).

## 4.2 Derivatives

### Definitions
- **Derivative**: f'(x) = lim(h->0) [f(x+h) - f(x)] / h
- Represents the instantaneous rate of change / slope of the tangent line
- f is **differentiable** at a if f'(a) exists

### Key Formulas — Differentiation Rules
- **Power Rule**: d/dx [x^n] = n*x^(n-1)
- **Constant Multiple**: d/dx [c*f(x)] = c*f'(x)
- **Sum/Difference**: d/dx [f +/- g] = f' +/- g'
- **Product Rule**: d/dx [f*g] = f'g + fg'
- **Quotient Rule**: d/dx [f/g] = (f'g - fg') / g^2
- **Chain Rule**: d/dx [f(g(x))] = f'(g(x)) * g'(x)

### Key Formulas — Common Derivatives
- d/dx [sin x] = cos x
- d/dx [cos x] = -sin x
- d/dx [tan x] = sec^2 x
- d/dx [e^x] = e^x
- d/dx [ln x] = 1/x
- d/dx [a^x] = a^x * ln(a)

### Key Terms
Tangent line, normal line, differentiable, critical point, derivative notation (f', dy/dx, Df)

### Worked Example
Find f'(x) for f(x) = 3x^4 - 2x^2 + 5x - 1.
f'(x) = 12x^3 - 4x + 5 (apply power rule to each term; derivative of constant is 0).

### Common Misconceptions
- "Differentiable implies continuous" is TRUE. But "continuous implies differentiable" is FALSE (e.g., |x| at x=0).
- Forgetting the chain rule when differentiating compositions like sin(3x). The answer is 3cos(3x), not just cos(3x).

## 4.3 Applications of Derivatives

### Definitions
- **Critical point**: where f'(x) = 0 or f'(x) is undefined
- **Local max/min**: identified by First or Second Derivative Test
- **Absolute (global) max/min**: largest/smallest values on a closed interval (check critical points AND endpoints)
- **Concavity**: f''(x) > 0 means concave up; f''(x) < 0 means concave down
- **Inflection point**: where concavity changes (f'' changes sign)
- **Mean Value Theorem**: If f is continuous on [a,b] and differentiable on (a,b), there exists c in (a,b) where f'(c) = [f(b)-f(a)]/(b-a)

### Optimization Procedure
1. Identify the quantity to optimize and write a formula
2. Express the formula in one variable using constraints
3. Find critical points (set derivative = 0)
4. Verify max/min using First or Second Derivative Test
5. Answer in context

### Related Rates Procedure
1. Draw a diagram and label variables
2. Write an equation relating the variables
3. Differentiate both sides with respect to time t
4. Substitute known values and solve

### Key Terms
Increasing/decreasing function, first derivative test, second derivative test, related rates, optimization, concavity, inflection point

### Worked Example — Optimization
A farmer has 200 ft of fencing. Find the dimensions of the rectangular pen of maximum area.
Let x = width, y = length. Constraint: 2x + 2y = 200, so y = 100 - x.
A = xy = x(100-x) = 100x - x^2.
A' = 100 - 2x = 0 => x = 50. y = 50.
A'' = -2 < 0, confirming maximum. Max area = 2500 sq ft with a 50x50 square.

### Common Misconceptions
- Forgetting to check endpoints for absolute extrema on closed intervals.
- Assuming f'(c) = 0 always means a local extremum (could be an inflection point if f'' also equals 0).

## 4.4 Integrals (Introduction)

### Definitions
- **Antiderivative**: F is an antiderivative of f if F'(x) = f(x)
- **Indefinite integral**: integral of f(x) dx = F(x) + C
- **Definite integral**: integral from a to b of f(x) dx = F(b) - F(a) [Fundamental Theorem of Calculus]
- **Riemann Sum**: Approximation of area using rectangles (left, right, midpoint)

### Key Formulas
- integral of x^n dx = x^(n+1)/(n+1) + C, n != -1
- integral of 1/x dx = ln|x| + C
- integral of e^x dx = e^x + C
- integral of sin x dx = -cos x + C
- integral of cos x dx = sin x + C
- **Fundamental Theorem of Calculus Part 1**: d/dx [integral from a to x of f(t) dt] = f(x)
- **Fundamental Theorem of Calculus Part 2**: integral from a to b of f(x) dx = F(b) - F(a)

### Key Terms
Integrand, limits of integration, net area, total area, u-substitution, area between curves

### Worked Example
Evaluate the integral from 0 to 3 of (2x + 1) dx.
Antiderivative: x^2 + x. Evaluate: (9 + 3) - (0 + 0) = 12.

### Common Misconceptions
- Forgetting the +C on indefinite integrals.
- Confusing net area (can be negative) with total area (always positive — use absolute value of integrand).

# 5. SUBJECT KNOWLEDGE — CALCULUS II

**Course**: MATH 122 — Calculus and Analytic Geometry II

## 5.1 Integration Techniques

### Methods
1. **u-Substitution**: integral of f(g(x))*g'(x) dx = integral of f(u) du where u = g(x)
2. **Integration by Parts**: integral of u dv = uv - integral of v du (LIATE rule for choosing u: Logarithmic, Inverse trig, Algebraic, Trig, Exponential)
3. **Trigonometric Integrals**: Use identities (sin^2 + cos^2 = 1, double-angle formulas)
4. **Trigonometric Substitution**: For sqrt(a^2 - x^2), let x = a*sin(theta); for sqrt(a^2 + x^2), let x = a*tan(theta); for sqrt(x^2 - a^2), let x = a*sec(theta)
5. **Partial Fractions**: Decompose rational functions into simpler fractions before integrating

### Worked Example — Integration by Parts
integral of x*e^x dx. Let u = x, dv = e^x dx. Then du = dx, v = e^x.
= x*e^x - integral of e^x dx = x*e^x - e^x + C = e^x(x - 1) + C.

### Common Misconceptions
- Applying u-substitution without adjusting dx. If u = 3x, then du = 3dx, so dx = du/3.
- Forgetting to convert back to x after trig substitution.

## 5.2 Improper Integrals

### Definition
- Integrals with infinite limits or discontinuous integrands
- **Convergent** if the limit exists and is finite; **divergent** otherwise

### Key Formulas
- integral from 1 to inf of 1/x^p dx converges if p > 1, diverges if p <= 1 (p-test)
- Comparison test: compare to known convergent/divergent integral

## 5.3 Sequences and Series

### Definitions
- **Sequence**: ordered list {a_n} with terms a_1, a_2, a_3, ...
- **Series**: sum of sequence terms: S = sum(n=1 to inf) a_n
- **Partial sum**: S_N = sum(n=1 to N) a_n
- **Convergent series**: lim(N->inf) S_N exists and is finite

### Convergence Tests
1. **Divergence Test (nth-term test)**: If lim a_n != 0, the series diverges. (Converse is NOT true!)
2. **Geometric Series**: sum(ar^n) converges to a/(1-r) if |r| < 1
3. **p-Series**: sum(1/n^p) converges if p > 1
4. **Integral Test**: Compare series to integral; both converge or both diverge
5. **Comparison Test**: Compare to known convergent/divergent series
6. **Limit Comparison Test**: lim(a_n/b_n) = L (0 < L < inf) => both converge or both diverge
7. **Alternating Series Test**: If a_n decreasing and lim a_n = 0, alternating series converges
8. **Ratio Test**: lim |a_(n+1)/a_n| = L. Converges if L < 1, diverges if L > 1
9. **Root Test**: lim |a_n|^(1/n) = L. Same criteria as Ratio Test

### Key Terms
Absolute convergence, conditional convergence, radius of convergence, interval of convergence

### Common Misconceptions
- "If the nth term goes to 0, the series converges" — FALSE (harmonic series 1/n diverges).
- Confusing sequence convergence with series convergence.

## 5.4 Taylor and Maclaurin Series

### Definitions
- **Taylor series** of f(x) centered at a: sum(n=0 to inf) f^(n)(a)/n! * (x-a)^n
- **Maclaurin series**: Taylor series centered at a = 0

### Important Maclaurin Series
- e^x = sum(x^n/n!) = 1 + x + x^2/2! + x^3/3! + ...
- sin x = sum((-1)^n * x^(2n+1)/(2n+1)!) = x - x^3/3! + x^5/5! - ...
- cos x = sum((-1)^n * x^(2n)/(2n)!) = 1 - x^2/2! + x^4/4! - ...
- 1/(1-x) = sum(x^n) = 1 + x + x^2 + ... for |x| < 1
- ln(1+x) = sum((-1)^(n+1) * x^n/n) = x - x^2/2 + x^3/3 - ... for |x| <= 1

## 5.5 L'Hopital's Rule

### Statement
If lim f(x)/g(x) is indeterminate (0/0 or inf/inf), then lim f(x)/g(x) = lim f'(x)/g'(x), provided the latter exists.

### Worked Example
lim(x->0) sin(x)/x. Form 0/0. Apply L'Hopital: lim cos(x)/1 = 1.

### Common Misconceptions
- Applying L'Hopital's Rule when the form is NOT indeterminate.
- Using the quotient rule instead of differentiating numerator and denominator separately.

# 6. SUBJECT KNOWLEDGE — LINEAR ALGEBRA

**Course**: MATH 226 — Linear Algebra

## 6.1 Systems of Linear Equations

### Definitions
- **System of linear equations**: set of equations each linear in the unknowns
- **Augmented matrix**: [A | b] representing the system Ax = b
- **Row echelon form (REF)**: leading 1 in each row, zeros below pivots
- **Reduced row echelon form (RREF)**: REF with zeros above and below pivots

### Key Methods
- **Gaussian elimination**: row operations to reach REF, then back-substitute
- **Gauss-Jordan elimination**: row operations to reach RREF (direct solution)
- **Row operations**: swap rows, multiply row by nonzero scalar, add scalar multiple of one row to another

### Solution Types
- **Unique solution**: same number of pivots as unknowns
- **No solution** (inconsistent): a row like [0 0 ... 0 | b] with b != 0
- **Infinitely many solutions**: free variables exist (fewer pivots than unknowns)

## 6.2 Matrices

### Definitions
- **Matrix**: rectangular array of numbers (m x n = m rows, n columns)
- **Square matrix**: m = n
- **Identity matrix I**: square matrix with 1s on diagonal, 0s elsewhere
- **Inverse A^(-1)**: AA^(-1) = A^(-1)A = I (exists only if det(A) != 0)
- **Transpose A^T**: rows become columns

### Key Formulas
- **Matrix multiplication**: (AB)_{ij} = sum_k A_{ik} * B_{kj}. Requires columns of A = rows of B.
- **2x2 Inverse**: If A = [[a,b],[c,d]], then A^(-1) = (1/det(A)) * [[d,-b],[-c,a]]
- **Properties**: (AB)^T = B^T * A^T; (AB)^(-1) = B^(-1) * A^(-1)

### Common Misconceptions
- "AB = BA" — Matrix multiplication is NOT commutative in general.
- "If AB = 0, then A = 0 or B = 0" — FALSE for matrices.

## 6.3 Determinants

### Definitions
- **Determinant**: scalar value computed from a square matrix
- det(A) = 0 means A is singular (not invertible)
- det(A) != 0 means A is invertible

### Key Formulas
- **2x2**: det([[a,b],[c,d]]) = ad - bc
- **3x3**: Cofactor expansion along any row or column
- **Properties**: det(AB) = det(A)*det(B); det(A^T) = det(A); det(kA) = k^n * det(A) for n x n matrix

## 6.4 Vector Spaces

### Definitions
- **Vector space**: a set V with addition and scalar multiplication satisfying 10 axioms
- **Subspace**: nonempty subset of V closed under addition and scalar multiplication (must contain zero vector)
- **Span**: set of all linear combinations of a set of vectors
- **Linear independence**: no vector in the set is a linear combination of the others
- **Basis**: linearly independent set that spans V
- **Dimension**: number of vectors in a basis

### Key Terms
Null space (kernel), column space (range), row space, rank, nullity

### Key Theorem
**Rank-Nullity Theorem**: rank(A) + nullity(A) = n (number of columns)

## 6.5 Eigenvalues and Eigenvectors

### Definitions
- **Eigenvalue** lambda: scalar such that Av = lambda*v for some nonzero v
- **Eigenvector** v: nonzero vector satisfying Av = lambda*v
- **Characteristic equation**: det(A - lambda*I) = 0
- **Eigenspace**: set of all eigenvectors for a given eigenvalue (plus zero vector)

### Procedure
1. Solve det(A - lambda*I) = 0 for eigenvalues lambda
2. For each lambda, solve (A - lambda*I)v = 0 for eigenvectors

### Worked Example
A = [[2,1],[0,3]]. Characteristic equation: (2-lambda)(3-lambda) = 0.
Eigenvalues: lambda_1 = 2, lambda_2 = 3.
For lambda = 2: (A - 2I)v = 0 => [[0,1],[0,1]]v = 0 => v = t[1,0].
For lambda = 3: (A - 3I)v = 0 => [[-1,1],[0,0]]v = 0 => v = t[1,1].

### Key Terms
Diagonalization, similar matrices, characteristic polynomial, algebraic/geometric multiplicity

### Common Misconceptions
- "Every matrix is diagonalizable" — FALSE. Requires n linearly independent eigenvectors.
- The zero vector is NEVER an eigenvector (by definition).

## 7.1 Functions and Models (MATH 110)

### Key Function Types

- **Exponential**: f(x) = a*b^x (growth if b > 1, decay if 0 < b < 1)

## 7.2 Marginal Analysis

### Definitions
- **Marginal cost**: MC = C'(x) — rate of change of total cost
- **Marginal revenue**: MR = R'(x) — rate of change of total revenue
- **Marginal profit**: MP = P'(x) = R'(x) - C'(x)
- **Break-even point**: where R(x) = C(x) or P(x) = 0

### Key Insight
"Marginal" means "the derivative of." Marginal cost at x = 100 approximates the cost of producing the 101st unit.

### Worked Example
C(x) = 0.5x^2 + 10x + 100. Find MC at x = 50.
MC = C'(x) = x + 10. MC(50) = 60. The 51st unit costs approximately $60.

## 7.3 Optimization in Business

### Profit Maximization
- Set P'(x) = 0 and solve. Verify P''(x) < 0 (maximum).
- Equivalently, set MR = MC (marginal revenue = marginal cost).

### Revenue Maximization
- R(x) = p*x where p = D(x) is the demand function
- Set R'(x) = 0 and solve

### Elasticity of Demand
- E = (p/q) * (dq/dp)
- |E| > 1: elastic (revenue increases when price decreases)
- |E| < 1: inelastic (revenue increases when price increases)
- |E| = 1: unit elastic (revenue maximized)

## 7.4 Mathematics of Finance (MATH 129)

### Simple Interest
- I = Prt (Interest = Principal * rate * time)
- A = P(1 + rt) (Future value)

### Compound Interest
- A = P(1 + r/n)^(nt) (n = compounding periods per year)
- A = Pe^(rt) (continuous compounding)
- **Effective annual rate**: r_eff = (1 + r/n)^n - 1

### Annuities
- **Ordinary annuity** (payments at end of period):
  - FV = PMT * [(1 + r/n)^(nt) - 1] / (r/n)
  - PV = PMT * [1 - (1 + r/n)^(-nt)] / (r/n)
- **Annuity due** (payments at beginning): multiply ordinary annuity formulas by (1 + r/n)

### Amortization
- PMT = PV * (r/n) / [1 - (1 + r/n)^(-nt)]
- Amortization schedule: each payment splits into interest and principal portions

### Linear Programming
- Objective function: maximize or minimize Z = ax + by
- Subject to constraint inequalities
- Solution at vertex (corner point) of feasible region
- **Procedure**: Graph constraints, identify feasible region, test vertices in objective function

### Worked Example — Compound Interest
Invest $5000 at 6% compounded monthly for 10 years.
A = 5000(1 + 0.06/12)^(12*10) = 5000(1.005)^120 = 5000(1.8194) = $9,096.98.

# 8. SUBJECT KNOWLEDGE — STATISTICS I

**Course**: MATH 201 — Statistics I

## 8.1 Descriptive Statistics

### Measures of Center
- **Mean (x-bar)**: sum of all values / n. Affected by outliers.
- **Median**: middle value when sorted. Resistant to outliers.
- **Mode**: most frequent value. Can be multimodal.

### Measures of Spread
- **Range**: max - min
- **Variance (s^2)**: s^2 = sum((x_i - x-bar)^2) / (n-1) [sample]
- **Standard Deviation (s)**: s = sqrt(s^2)
- **Interquartile Range (IQR)**: Q3 - Q1
- **Coefficient of Variation**: CV = (s / x-bar) * 100%

### Five-Number Summary
Minimum, Q1, Median, Q3, Maximum. Used for boxplots.

### Key Terms
Outlier (typically > Q3 + 1.5*IQR or < Q1 - 1.5*IQR), skewness (left-skewed: mean < median; right-skewed: mean > median), frequency distribution, histogram, stem-and-leaf plot

### Worked Example
Data: 3, 7, 7, 12, 15. Mean = (3+7+7+12+15)/5 = 8.8. Median = 7. Mode = 7.
Variance = [(3-8.8)^2 + (7-8.8)^2 + (7-8.8)^2 + (12-8.8)^2 + (15-8.8)^2] / 4
= [33.64 + 3.24 + 3.24 + 10.24 + 38.44] / 4 = 88.8 / 4 = 22.2.
SD = sqrt(22.2) = 4.71.

### Common Misconceptions
- Using n instead of (n-1) for sample variance (Bessel's correction).
- Thinking the mean is always the best measure of center (use median for skewed data).

## 8.2 Probability

### Basic Rules
- **Complement**: P(A') = 1 - P(A)
- **Addition Rule**: P(A or B) = P(A) + P(B) - P(A and B)
- **Mutually exclusive**: P(A and B) = 0, so P(A or B) = P(A) + P(B)
- **Multiplication Rule**: P(A and B) = P(A) * P(B|A)
- **Independent events**: P(A and B) = P(A) * P(B); P(B|A) = P(B)

### Conditional Probability
- P(A|B) = P(A and B) / P(B)

### Bayes' Theorem
P(A|B) = P(B|A) * P(A) / P(B)

More generally: P(A_i|B) = P(B|A_i)*P(A_i) / sum_j[P(B|A_j)*P(A_j)]

### Counting
- **Permutations**: P(n,r) = n! / (n-r)! (order matters)
- **Combinations**: C(n,r) = n! / [r!(n-r)!] (order doesn't matter)

### Worked Example — Bayes' Theorem
1% of population has a disease. Test is 95% accurate (true positive) and 90% specific (true negative).
P(Disease) = 0.01, P(+|Disease) = 0.95, P(+|No Disease) = 0.10.
P(Disease|+) = (0.95 * 0.01) / (0.95*0.01 + 0.10*0.99) = 0.0095 / 0.1085 = 0.0876 or about 8.8%.

### Common Misconceptions
- Confusing P(A|B) with P(B|A) ("prosecutor's fallacy").
- Assuming independence without justification.
- Forgetting to subtract P(A and B) in the addition rule when events are not mutually exclusive.

## 8.3 Probability Distributions

### Discrete Distributions

**Binomial Distribution**: X ~ Bin(n, p)
- P(X = k) = C(n,k) * p^k * (1-p)^(n-k)
- Mean: mu = np
- Variance: sigma^2 = np(1-p)
- Requirements: fixed n trials, two outcomes, constant p, independent trials

**Poisson Distribution**: X ~ Poisson(lambda)
- P(X = k) = e^(-lambda) * lambda^k / k!
- Mean = Variance = lambda
- Models rare events in a fixed interval

### Continuous Distributions

**Normal Distribution**: X ~ N(mu, sigma^2)
- Bell-shaped, symmetric about mu
- **Empirical Rule**: 68% within 1 SD, 95% within 2 SD, 99.7% within 3 SD
- **Standard normal**: Z = (X - mu) / sigma ~ N(0,1)
- Use Z-table or technology for probabilities

### Worked Example — Binomial
A fair coin is flipped 10 times. P(exactly 6 heads):
P(X=6) = C(10,6) * (0.5)^6 * (0.5)^4 = 210 * (1/1024) = 0.2051 or 20.5%.

### Worked Example — Normal
SAT scores ~ N(1060, 195^2). P(score > 1200):
Z = (1200 - 1060)/195 = 0.718. P(Z > 0.718) = 1 - 0.7637 = 0.2363 or 23.6%.

### Common Misconceptions
- Using the binomial formula when trials are not independent.
- Forgetting continuity correction when approximating discrete with normal.

## 8.4 Confidence Intervals

### Definition
A range of values, constructed from sample data, that is likely to contain the true population parameter with a specified confidence level.

### Formulas
- **For population mean (sigma known)**: x-bar +/- z*(sigma/sqrt(n))
- **For population mean (sigma unknown)**: x-bar +/- t*(s/sqrt(n)) [use t-distribution with df = n-1]
- **For population proportion**: p-hat +/- z*sqrt(p-hat*(1-p-hat)/n)

### Key Terms
- **Margin of error (E)**: the "+/-" part of the interval
- **Confidence level**: probability that the interval contains the true parameter (commonly 90%, 95%, 99%)
- **Critical values**: z* = 1.645 (90%), 1.96 (95%), 2.576 (99%)

### Sample Size Determination
- For mean: n = (z*sigma/E)^2
- For proportion: n = (z/E)^2 * p-hat*(1-p-hat); use p-hat = 0.5 if unknown

### Worked Example
Sample: n=36, x-bar=52, s=6. Find 95% CI for mu.
Since n >= 30, use z*: 52 +/- 1.96*(6/sqrt(36)) = 52 +/- 1.96 = (50.04, 53.96).

### Common Misconceptions
- "95% confidence means there's a 95% probability the true mean is in THIS interval" — the parameter is fixed; the confidence refers to the method's long-run success rate.
- Wider confidence level = wider interval (not more precise).

## 8.5 Hypothesis Testing

### Procedure
1. State H_0 (null hypothesis) and H_a (alternative hypothesis)
2. Choose significance level alpha (commonly 0.05)
3. Compute test statistic
4. Find p-value or compare to critical value
5. Make decision: reject H_0 if p-value < alpha
6. State conclusion in context

### Test Statistics
- **Z-test for mean (sigma known)**: z = (x-bar - mu_0) / (sigma/sqrt(n))
- **t-test for mean (sigma unknown)**: t = (x-bar - mu_0) / (s/sqrt(n)), df = n-1
- **Z-test for proportion**: z = (p-hat - p_0) / sqrt(p_0*(1-p_0)/n)

### Types of Errors
- **Type I error (alpha)**: rejecting H_0 when it is true (false positive)
- **Type II error (beta)**: failing to reject H_0 when it is false (false negative)
- **Power**: 1 - beta (probability of correctly rejecting a false H_0)

### One-tail vs Two-tail
- H_a: mu != mu_0 (two-tailed)
- H_a: mu > mu_0 (right-tailed)
- H_a: mu < mu_0 (left-tailed)

### Worked Example
Claim: mean weight = 50 kg. Sample: n=25, x-bar=52, s=5. Test at alpha=0.05.
H_0: mu = 50. H_a: mu != 50.
t = (52-50)/(5/sqrt(25)) = 2/1 = 2.0. df = 24. Critical t = +/-2.064 (two-tailed, 0.05).
|t| = 2.0 < 2.064, so fail to reject H_0. (Or p-value approx 0.057 > 0.05.)

### Common Misconceptions
- "Fail to reject H_0" does NOT mean "H_0 is true." It means insufficient evidence.
- A small p-value indicates statistical significance, not necessarily practical significance.

## 8.6 Simple Linear Regression

### Model
y = beta_0 + beta_1*x + epsilon

### Estimated Regression Line
y-hat = b_0 + b_1*x

### Formulas
- b_1 = sum((x_i - x-bar)(y_i - y-bar)) / sum((x_i - x-bar)^2) = r*(s_y/s_x)
- b_0 = y-bar - b_1*x-bar
- **Correlation coefficient r**: measures linear association (-1 <= r <= 1)
- **Coefficient of determination R^2**: proportion of variance in y explained by x. R^2 = r^2 for simple regression.

### Key Terms
Residual (e_i = y_i - y-hat_i), least squares method, scatter plot, interpolation vs extrapolation

### Assumptions (LINE)
- **L**inearity
- **I**ndependence of errors
- **N**ormality of errors
- **E**qual variance (homoscedasticity)

### Common Misconceptions
- "Correlation implies causation" — FALSE. Always consider lurking variables.
- Extrapolating far beyond the data range is unreliable.

## 9.1 Multiple Regression

### Model
y = beta_0 + beta_1*x_1 + beta_2*x_2 + ... + beta_k*x_k + epsilon

### Key Concepts
- **Adjusted R^2**: accounts for number of predictors; penalizes adding weak predictors
  - R^2_adj = 1 - [(1 - R^2)(n-1)/(n-k-1)]
- **F-test for overall significance**: tests if at least one predictor is significant
  - H_0: all beta_i = 0; H_a: at least one beta_i != 0
- **t-test for individual coefficients**: tests if a single predictor is significant
- **Multicollinearity**: when predictors are highly correlated with each other
  - Detected by VIF (Variance Inflation Factor); VIF > 10 is problematic
- **Dummy variables**: represent categorical predictors (k categories need k-1 dummies)

### Model Building
- Forward selection, backward elimination, stepwise regression
- AIC and BIC criteria for model comparison (lower is better)

## 9.2 ANOVA (Analysis of Variance)

### Purpose
Test whether means of 3 or more groups are equal.

### One-Way ANOVA
- H_0: mu_1 = mu_2 = ... = mu_k
- H_a: at least one mean differs
- **F-statistic**: F = MSB / MSW (Mean Square Between / Mean Square Within)
  - MSB = SSB / (k-1)
  - MSW = SSW / (N-k)
- Reject H_0 if F > F_critical or p-value < alpha

### ANOVA Table
| Source | SS | df | MS | F |
|--------|----|----|----|---|
| Between | SSB | k-1 | SSB/(k-1) | MSB/MSW |
| Within | SSW | N-k | SSW/(N-k) | |
| Total | SST | N-1 | | |

### Assumptions
1. Independence of observations
2. Normality within each group
3. Equal variances (homogeneity) — check with Levene's test

### Post-Hoc Tests
After rejecting H_0, determine WHICH means differ:
- **Tukey's HSD**: controls family-wise error rate for all pairwise comparisons
- **Bonferroni correction**: divide alpha by number of comparisons
- **Scheffe's method**: most conservative, allows complex contrasts

### Two-Way ANOVA
- Tests effects of two factors and their interaction
- Main effect A, main effect B, interaction A*B
- **Interaction**: when the effect of one factor depends on the level of the other

### Common Misconceptions
- "Significant ANOVA F-test tells you which groups differ" — NO, you need post-hoc tests.
- Violating homogeneity assumption can inflate Type I error rate.

## 9.3 Chi-Square Tests

### Chi-Square Goodness of Fit
- Tests whether observed frequencies match expected frequencies
- chi^2 = sum((O_i - E_i)^2 / E_i), df = k - 1
- H_0: data follows the specified distribution

### Chi-Square Test of Independence
- Tests whether two categorical variables are related
- chi^2 = sum((O - E)^2 / E) where E = (row total * column total) / grand total
- df = (r-1)(c-1) where r = rows, c = columns
- H_0: variables are independent

### Requirements
- Expected frequency >= 5 in each cell (or at least 80% of cells)

### Worked Example — Independence
Testing if gender and preference (A/B/C) are independent:

|  | A | B | C | Total |
|--|---|---|---|-------|
| M | 30 | 20 | 50 | 100 |
| F | 40 | 30 | 30 | 100 |
| Total | 70 | 50 | 80 | 200 |

Expected for M-A: 100*70/200 = 35. chi^2 contribution: (30-35)^2/35 = 0.714.
Continue for all cells, sum chi^2, compare to critical value with df = (2-1)(3-1) = 2.

### Common Misconceptions
- Chi-square tests work on counts/frequencies, NOT means or percentages directly.
- Small expected frequencies violate the test's assumptions.

## 9.4 Nonparametric Statistics

### When to Use
- Data violate normality assumption
- Ordinal data
- Small sample sizes with unknown distributions

### Common Tests
- **Mann-Whitney U test**: nonparametric alternative to independent two-sample t-test
- **Wilcoxon Signed-Rank test**: nonparametric alternative to paired t-test
- **Kruskal-Wallis test**: nonparametric alternative to one-way ANOVA
- **Spearman's rank correlation**: nonparametric alternative to Pearson's r

## 10.2 Actuarial Statistics II (AM 231)

### Transformation of Variables
- If Y = g(X) and g is monotone: f_Y(y) = f_X(g^(-1)(y)) * |d/dy[g^(-1)(y)]|

### Central Limit Theorem
For large n, the sum (or mean) of iid random variables is approximately normal, regardless of the original distribution.

### Estimation

- **Method of Moments**: set sample moments equal to population moments and solve.
- **Properties of estimators**: unbiased (E[theta-hat] = theta), consistent, efficient, sufficient

### Hypothesis Testing (Actuarial Context)
- Likelihood ratio tests
- Neyman-Pearson Lemma: most powerful test for simple hypotheses

- **Atomic number (Z)**: number of protons (defines element)
- **Mass number (A)**: protons + neutrons
- **Isotopes**: same element, different number of neutrons
- **Electron configuration**: Aufbau principle, Pauli exclusion, Hund's rule
- **Quantum numbers**: n (principal), l (angular), m_l (magnetic), m_s (spin)

### Periodic Table Trends
- **Atomic radius**: increases down a group, decreases across a period
- **Ionization energy**: decreases down a group, increases across a period
- **Electronegativity**: decreases down a group, increases across a period (Pauling scale)
- **Electron affinity**: generally increases across a period

### Chemical Bonding
- **Ionic bonds**: transfer of electrons (metal + nonmetal); form crystal lattices
- **Covalent bonds**: sharing of electrons (nonmetals); polar vs nonpolar
- **Metallic bonds**: "sea of electrons" model
- **Lewis structures**: show valence electrons and bonding
- **VSEPR theory**: predicts molecular geometry (linear, trigonal planar, tetrahedral, etc.)

### Chemical Reactions
- **Balancing equations**: conservation of mass
- **Types**: synthesis, decomposition, single replacement, double replacement, combustion
- **Stoichiometry**: mole ratios from balanced equations
- **Mole concept**: 1 mol = 6.022 x 10^23 particles (Avogadro's number)
- **Molar mass**: mass of 1 mole of a substance (g/mol)
- **Limiting reagent**: reactant that runs out first; determines theoretical yield
- **Percent yield**: (actual yield / theoretical yield) x 100%

### States of Matter
- **Gas laws**: Boyle's (PV=k), Charles's (V/T=k), Avogadro's (V/n=k), Ideal Gas Law (PV=nRT)
- **Solutions**: solute + solvent; concentration units (molarity M = mol/L)

## 13.2 Introductory Chemistry II (SCI 267)

### Chemical Equilibrium
- **Equilibrium constant K**: K = [products]^coefficients / [reactants]^coefficients
- **Le Chatelier's Principle**: system shifts to counteract a stress
- **Reaction quotient Q**: compare to K to predict direction of shift

### Acids and Bases
- **Arrhenius**: acids produce H+; bases produce OH-
- **Bronsted-Lowry**: acids donate H+; bases accept H+
- **pH scale**: pH = -log[H+]; pH 7 neutral, <7 acidic, >7 basic
- **Strong acids/bases**: fully dissociate (HCl, NaOH, etc.)
- **Weak acids/bases**: partially dissociate; Ka, Kb values
- **Buffer solutions**: resist pH change; Henderson-Hasselbalch: pH = pKa + log([A-]/[HA])

### Thermodynamics
- **Enthalpy (delta H)**: heat of reaction; exothermic (<0) vs endothermic (>0)
- **Entropy (delta S)**: measure of disorder; tends to increase
- **Gibbs free energy**: delta G = delta H - T*delta S; spontaneous if delta G < 0

### Kinetics
- **Rate law**: Rate = k[A]^m[B]^n (determined experimentally)
- **Activation energy**: minimum energy for reaction (Arrhenius equation: k = Ae^(-Ea/RT))
- **Catalysts**: lower activation energy without being consumed

### Electrochemistry
- **Oxidation**: loss of electrons; **Reduction**: gain of electrons (OIL RIG)
- **Galvanic (voltaic) cells**: spontaneous redox; produce electricity
- **Electrolytic cells**: non-spontaneous; require external power
- **Standard cell potential**: E_cell = E_cathode - E_anode

# 14. SUBJECT KNOWLEDGE — PHYSICS

**Course**: SCI 264 — Physics I: Introductory Physics

## 14.1 Mechanics

### Kinematics
- **Displacement**: change in position (vector)
- **Velocity**: rate of change of displacement; v = dx/dt
- **Acceleration**: rate of change of velocity; a = dv/dt
- **Kinematic equations** (constant acceleration):
  - v = v_0 + at
  - x = x_0 + v_0*t + (1/2)at^2
  - v^2 = v_0^2 + 2a(x - x_0)

### Newton's Laws
1. **First Law** (Inertia): object at rest stays at rest; object in motion stays in motion unless acted on by net external force
2. **Second Law**: F_net = ma
3. **Third Law**: every action has an equal and opposite reaction

### Forces
- **Weight**: W = mg (g = 9.8 m/s^2)
- **Normal force**: perpendicular contact force from surface
- **Friction**: f = mu*N (static: f_s <= mu_s*N; kinetic: f_k = mu_k*N)
- **Tension**: force transmitted through a rope/string/cable

### Energy and Work
- **Work**: W = F*d*cos(theta)
- **Kinetic energy**: KE = (1/2)mv^2
- **Potential energy**: PE = mgh (gravitational); PE = (1/2)kx^2 (elastic)
- **Conservation of energy**: KE_1 + PE_1 = KE_2 + PE_2 (if no non-conservative forces)
- **Power**: P = W/t = F*v

### Momentum
- **Momentum**: p = mv
- **Impulse**: J = F*delta_t = delta_p
- **Conservation of momentum**: total momentum is conserved in isolated systems
- **Elastic collision**: both momentum and KE conserved
- **Inelastic collision**: momentum conserved; KE not conserved

## 14.2 Waves and Sound
- **Wave types**: transverse (perpendicular oscillation) vs longitudinal (parallel oscillation, e.g., sound)
- **Wave equation**: v = f*lambda (velocity = frequency * wavelength)
- **Sound**: longitudinal mechanical wave; speed in air ~343 m/s
- **Doppler effect**: frequency change due to relative motion of source and observer

## 14.3 Heat and Thermodynamics
- **Temperature scales**: Celsius, Fahrenheit, Kelvin (K = C + 273.15)
- **Heat transfer**: conduction, convection, radiation
- **Specific heat**: Q = mc*delta_T
- **Laws of thermodynamics**:
  - 0th: thermal equilibrium is transitive
  - 1st: delta_U = Q - W (conservation of energy)
  - 2nd: entropy of isolated system never decreases
  - 3rd: entropy approaches zero as temperature approaches absolute zero

## 14.4 Electricity
- **Charge**: positive and negative; like charges repel, opposite attract
- **Coulomb's Law**: F = kq_1q_2/r^2 (k = 8.99 x 10^9 N*m^2/C^2)
- **Electric field**: E = F/q = kQ/r^2
- **Voltage**: V = W/q (energy per charge)
- **Ohm's Law**: V = IR
- **Power**: P = IV = I^2R = V^2/R
- **Series circuits**: R_total = R1 + R2 + ...; same current
- **Parallel circuits**: 1/R_total = 1/R1 + 1/R2 + ...; same voltage

## 14.5 Light and Optics
- **Electromagnetic spectrum**: radio, microwave, infrared, visible, UV, X-ray, gamma
- **Speed of light**: c = 3.0 x 10^8 m/s
- **Reflection**: angle of incidence = angle of reflection
- **Refraction**: Snell's Law: n_1*sin(theta_1) = n_2*sin(theta_2)
- **Lenses**: converging (convex) and diverging (concave); thin lens equation: 1/f = 1/d_o + 1/d_i

# APPENDIX C: COMMON MISCONCEPTIONS SUMMARY

| Topic | Misconception | Truth |
|-------|--------------|-------|
| Limits | f(c) undefined means limit DNE | Limit depends on nearby behavior |
| Derivatives | Continuous implies differentiable | Continuous does NOT imply differentiable |
| Series | nth term -> 0 means convergence | Harmonic series is counterexample |
| Matrices | AB = BA | Matrix multiplication is not commutative |
| Eigenvalues | Every matrix is diagonalizable | Requires n independent eigenvectors |
| Stats | Correlation implies causation | Correlation is not causation |
| Stats | Fail to reject = H_0 is true | Means insufficient evidence |
| CI | 95% prob the mean is in this interval | Confidence refers to the method |
| Regression | Can extrapolate freely | Extrapolation beyond data is unreliable |
| Probability | P(A\|B) = P(B\|A) | These are generally different |
| Chi-square | Works on percentages | Works on counts/frequencies |
| ANOVA | F-test tells which groups differ | Need post-hoc tests |
