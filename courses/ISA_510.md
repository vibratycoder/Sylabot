### Probability theory

Probability theory is the rigorous mathematical framework for analyzing random phenomena and quantifying uncertainty. It extends basic probability concepts into a formal system of axioms, theorems, and models that underpin statistical inference, stochastic processes, and machine learning algorithms.

- **Key concepts:** Probability spaces, sigma-algebras, random variables, probability mass/density functions, cumulative distribution functions, joint and marginal distributions
- **Important distributions:** Normal, binomial, Poisson, exponential, chi-squared, Student's t, F-distribution; each serves specific modeling purposes
- **Expectation and variance:** E[X] measures the center of a distribution; Var(X) measures spread; moment-generating functions characterize distributions uniquely
- **Limit theorems:** Law of Large Numbers (sample averages converge to expected value), Central Limit Theorem (sums of independent random variables tend toward normal distribution)
- **Practical relevance:** Probability theory is essential for building predictive models, understanding sampling distributions, computing confidence intervals, and performing Bayesian analysis. It provides the theoretical justification for why statistical methods work.

### Statistical inference

Statistical inference is the process of drawing conclusions about population parameters based on sample data. It bridges the gap between observed data and the broader population, enabling analysts to make generalizations, predictions, and decisions with quantified uncertainty.

- **Key concepts:** Point estimation (sample mean, proportion), interval estimation (confidence intervals), estimator properties (unbiasedness, consistency, efficiency)
- **Methods of estimation:** Maximum likelihood estimation (MLE), method of moments, Bayesian estimation with prior distributions
- **Confidence intervals:** A 95% CI means that if sampling were repeated many times, approximately 95% of intervals would contain the true parameter
- **Bayesian vs. frequentist:** Frequentist inference treats parameters as fixed and data as random; Bayesian inference treats parameters as random with prior distributions updated by data
- **Practical relevance:** Statistical inference enables data analysts to determine whether observed patterns are genuine or due to chance, estimate population characteristics from surveys, and quantify the precision of model predictions. It is the foundation of evidence-based decision-making.

### Hypothesis testing

Hypothesis testing is a formal statistical procedure for making decisions about population parameters based on sample data. It provides a structured framework for evaluating claims, comparing groups, and determining whether observed effects are statistically significant or attributable to random variation.

- **Key concepts:** Null hypothesis (H0) vs. alternative hypothesis (H1), test statistic, p-value, significance level (alpha, typically 0.05), rejection region
- **Common tests:** t-test (comparing means), chi-square test (categorical associations), ANOVA (comparing multiple group means), z-test (large sample proportions)
- **Error types:** Type I error (false positive, rejecting true H0), Type II error (false negative, failing to reject false H0), statistical power (probability of correctly rejecting false H0)
- **Effect size and practical significance:** Cohen's d, R-squared, and odds ratios measure the magnitude of effects beyond just statistical significance
- **Practical relevance:** Hypothesis testing is central to A/B testing in business, clinical trials in healthcare, quality control in manufacturing, and model comparison in data science. Understanding its limitations (p-hacking, multiple comparisons) is crucial for drawing valid conclusions.

### Regression analysis

Regression analysis is a statistical method for modeling the relationship between a dependent variable and one or more independent variables. It quantifies how changes in predictors are associated with changes in outcomes, making it one of the most widely used techniques in analytics for prediction and explanation.

- **Simple linear regression:** Models Y = b0 + b1*X + error; ordinary least squares (OLS) minimizes the sum of squared residuals to find the best-fit line
- **Multiple regression:** Extends to multiple predictors; coefficients represent the effect of each predictor while controlling for others; adjusted R-squared accounts for model complexity
- **Assumptions:** Linearity, independence of errors, homoscedasticity (constant variance), normality of residuals, no multicollinearity among predictors
- **Advanced variants:** Logistic regression (binary outcomes), polynomial regression (nonlinear relationships), ridge/lasso regression (regularization to prevent overfitting)
- **Practical relevance:** Regression is used across industries for sales forecasting, risk assessment, pricing models, and causal analysis. It serves as both a standalone predictive tool and a foundation for understanding more complex machine learning algorithms.

### Python for statistics

Python for statistics refers to the ecosystem of Python libraries and tools specifically designed for statistical computing, data manipulation, and quantitative analysis. Python has become the dominant language for applied statistics due to its rich library ecosystem and integration with machine learning frameworks.

- **Core libraries:** NumPy (numerical arrays, linear algebra), pandas (DataFrames, data manipulation), SciPy (statistical tests, distributions, optimization), statsmodels (regression, time series, hypothesis tests)
- **Visualization:** matplotlib (foundational plotting), seaborn (statistical visualization with built-in themes), plotly (interactive charts)
- **Key workflows:** Loading and cleaning data with pandas, computing descriptive statistics, fitting regression models with statsmodels, running hypothesis tests with SciPy, visualizing distributions with seaborn
- **Jupyter Notebooks:** Interactive computing environment ideal for exploratory data analysis, combining code, output, and narrative in a single document
- **Practical relevance:** Python's statistical libraries enable analysts to perform the full spectrum of statistical tasks without switching tools. The pandas + statsmodels + seaborn stack provides a complete environment for data exploration, modeling, and reporting that scales from simple analyses to production-grade pipelines.
