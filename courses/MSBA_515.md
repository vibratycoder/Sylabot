### Statistics review

A statistics review covers the essential statistical concepts that form the quantitative foundation for business analytics and data science. This foundational knowledge ensures students can interpret data correctly, evaluate evidence, and apply appropriate analytical methods throughout the MSBA program.

- **Descriptive statistics:** Measures of central tendency (mean, median, mode), measures of spread (standard deviation, variance, IQR, range), data distributions (normal, skewed, bimodal), percentiles and quartiles
- **Inferential statistics:** Sampling distributions, confidence intervals, hypothesis testing (t-tests, chi-square), p-values and significance levels, Type I and Type II errors
- **Correlation and regression:** Pearson and Spearman correlation coefficients, simple linear regression (interpreting slope, intercept, R-squared), residual analysis
- **Probability basics:** Conditional probability, Bayes' theorem, common distributions (normal, binomial, Poisson), expected value and variance
- **Practical relevance:** Statistical literacy is the gateway skill for all analytics coursework. Students who solidify these foundations can more easily grasp advanced topics like machine learning, experimental design, and predictive modeling that build directly on these concepts.

### Programming basics

Programming basics cover the fundamental concepts of writing code to manipulate data, automate tasks, and build analytical solutions. For MSBA students, this primarily focuses on Python as the dominant language in data analytics and data science.

- **Core concepts:** Variables and data types (integers, floats, strings, booleans), operators (arithmetic, comparison, logical), control flow (if/elif/else, for loops, while loops), functions (definition, parameters, return values)
- **Data structures:** Lists (ordered, mutable), dictionaries (key-value pairs), tuples (ordered, immutable), sets (unique values); understanding when to use each structure
- **Working with data:** Reading files (CSV, JSON, Excel), basic pandas operations (DataFrames, filtering, grouping, merging), NumPy arrays for numerical computation
- **Best practices:** Meaningful variable names, code commenting, modular design with functions, debugging strategies, using Jupyter Notebooks for iterative analysis
- **Practical relevance:** Programming skills enable analysts to work with datasets too large for spreadsheets, automate repetitive tasks, build reproducible analyses, and implement statistical and machine learning models. Even basic Python proficiency dramatically expands an analyst's capabilities.

### Visualization basics

Visualization basics cover the fundamental principles and tools for representing data graphically to reveal patterns, trends, and insights. Effective visualization is both a science (rooted in perception research) and an art (requiring design judgment and audience awareness).

- **Chart selection:** Bar charts (categorical comparisons), line charts (trends over time), scatter plots (relationships between variables), histograms (distributions), pie charts (proportions -- use sparingly), box plots (distribution summaries)
- **Design principles:** Maximize data-ink ratio, avoid chartjunk and 3D effects, use consistent and accessible color palettes, label axes clearly, provide context with titles and annotations
- **Tools overview:** Excel/Google Sheets (basic charts), Power BI (interactive dashboards), Tableau (drag-and-drop analytics), Python matplotlib/seaborn (programmatic visualization)
- **Common mistakes:** Truncated y-axes that exaggerate differences, dual y-axes that mislead, inappropriate chart types for the data, excessive color or decoration that distracts from the message
- **Practical relevance:** Visualization is often the first and last step of analysis -- used initially for exploration and finally for communication. Mastering visualization basics ensures that insights reach decision-makers in a clear, accurate, and compelling format.

### Communication skills

Communication skills for analytics professionals encompass the ability to translate technical findings into clear, actionable insights for diverse audiences. This includes written reports, oral presentations, data storytelling, and the interpersonal skills needed to collaborate effectively with business stakeholders.

- **Audience awareness:** Tailoring message complexity, detail level, and vocabulary to the audience (executives want high-level impact; technical teams want methodology details); the "so what?" test for every finding
- **Data storytelling:** Structuring presentations with context (why it matters), findings (what the data shows), and recommendations (what to do next); using visuals to support rather than replace narrative
- **Written communication:** Clear executive summaries, well-structured reports with logical flow, concise documentation of methodology and assumptions, effective use of appendices for technical details
- **Presentation skills:** Confident delivery, handling Q&A, managing technical depth for mixed audiences, using slide design principles (one message per slide, minimal text)
- **Practical relevance:** Technical skills alone do not drive business impact -- the ability to communicate findings persuasively is what converts analysis into action. Surveys consistently show that communication skills are among the top competencies employers seek in analytics hires, often ranking above specific technical abilities.
